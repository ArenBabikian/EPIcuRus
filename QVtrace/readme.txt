** Accessing, Installing and using QVtrace:

1- To request access to QVtrace, please contact QRA Corp (https://qracorp.com). You will receive an email with the shared folder.

2- Download the zip file under the shared folder.

3- Open the user guide 'QVtrace_user_guide.pdf' and follow Section 2 to install and Start QVtrace.

4- To restart QVtrace, run from command line: docker restart qvtrace

5- Follow Section3 and Section4 to learn how to use the QVtrace user interface. Follow Section5 to learn how to write constraints in QCT language.
